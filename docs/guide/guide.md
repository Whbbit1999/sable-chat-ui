# SableChat UI

## 使用技术栈

- 本组件库使用 `React` + `styled-component` + `Ts` 技术开发
- 包管理工具使用 `pnpm`
- 组件库文档由 `dumi2` 驱动
- 自动生成组件基础代码插件 `hygen`

## 说明

本组件库适用于 `React` 学习和组件库搭建学习。

## 新增组件说明

安装 hygen 插件

```bash
npm i -g hygen
```

新增组件

> 组件名称由大写字母开头

```bash
hygen component new <组件名称>
```
