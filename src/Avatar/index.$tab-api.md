# Api

参数如下：

| 属性名         | 说明         | 类型             | 默认值 |
| -------------- | ------------ | ---------------- | ------ |
| src            | 头像路径     | string           | -      |
| size           | 头像尺寸     | string           | 48px   |
| status         | 登录状态     | online / offline | string |
| statusIconSize | 登录状态大小 | string           | 8px    |
